# 🛍️ Retail Sales Data Analysis Dashboard

## 📊 Project Overview

This project focuses on analyzing business sales data to identify key revenue trends, high-performing product categories, and customer purchasing patterns.

The goal of this analysis is to simulate a real-world business scenario where data analysts help companies make data-driven decisions.

Using sales transaction data, a dashboard was created to visualize important business insights such as revenue distribution, category performance, and customer trends.

---

## 🎯 Objectives

The main objectives of this project were:

* Analyze revenue trends from sales data
* Identify top-performing product categories
* Compare revenue contribution by gender
* Understand monthly sales patterns
* Present insights through a client-ready dashboard

---

## 🛠️ Tools Used

* Google Sheets / Excel
* Data Visualization (Charts & Pivot Tables)
* GitHub for project documentation

---

## 📂 Dataset

Dataset used in this project contains retail sales transaction information including:

* Transaction ID
* Customer ID
* Product Category
* Gender
* Quantity Purchased
* Price per Unit
* Total Amount
* Date of Transaction

The dataset was cleaned and analyzed to generate business insights.

---

## 📈 Dashboard Insights

### 1️⃣ Revenue by Product Category

Electronics generated the highest revenue among all categories, followed by Clothing and Beauty products.

### 2️⃣ Revenue by Gender

Female customers contributed slightly more revenue compared to male customers.

### 3️⃣ Monthly Revenue Trend

Sales fluctuated across months, with certain months showing significantly higher revenue, indicating seasonal demand patterns.

---

## 💡 Business Recommendations

Based on the analysis:

• The business should focus more on **Electronics products** since they generate the highest revenue.

• Marketing campaigns can target **female customers**, as they contribute slightly higher revenue.

• Months with higher sales trends can be used for **product launches and promotions**.

• Lower-performing months should include **discount campaigns or marketing strategies** to boost sales.
